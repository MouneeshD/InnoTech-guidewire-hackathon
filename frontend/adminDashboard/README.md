# GigShield AI Admin Dashboard

Frontend-only React admin dashboard for monitoring users, claims, risk analytics, fraud alerts, and notifications.

## Run

```bash
npm install
npm run dev
```

## Tech

- React + Vite
- Tailwind CSS
- React Router
- Recharts

## Structure

- `src/components` reusable dashboard UI
- `src/pages` route-level pages
- `src/layout` sidebar and top navigation shell
- `src/data` mock platform data
