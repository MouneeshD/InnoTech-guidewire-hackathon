# GigShield AI Mobile App

React Native frontend for the GigShield AI parametric insurance platform.

## Run

```bash
npm install
expo start
```

## Structure

- `components/` reusable UI building blocks
- `data/` mock dashboard, policy, claims, and notification data
- `navigation/` app stack and tab navigation
- `screens/` feature screens
- `theme/` shared colors and spacing
